Ученые намерены определять коронавирус по голосу
Исследователи из Кембриджского университета разработали приложение для сбора данных о кашле и изменениях голоса и дыхания у пациентов с COVID-19. Это позволит создать алгоритмы, определяющие болезнь по этим параметрам. Приложение COVID-19 Sounds доступно для браузеров Chrome и Firefox, скоро появятся версии для смартфонов на базе Android и iOS.
Так как COVID-19 — респираторное заболевание, оно сопровождается изменениями в голосе и дыхании. С помощью приложения исследователи надеются собрать достаточно данных, чтобы обучить нейросеть по этим изменениям выявлять болезнь.
Приложению понадобятся основные демографические и медицинские данные пользователей, образцы их речи, дыхания и кашля, а также информация о том, есть ли у них положительный результат теста на коронавирус. Медицинских консультаций приложение не предоставляет — оно лишь собирает информацию для исследовательских целей.
В дальнейшем эти данные предполагается использовать для более точной диагностики и выявления осложнений. Кроме данных по COVID-19 исследователи надеются собрать информацию и по другим респираторным заболеваниям.
